Codeforces Round #752 (Div. 2) A~D

2022-11-15 Tutorial ， Codeforces

官方题解传送门

A

题意：在数组中添加任意数字，使得 ai≤ia_i \le iai​≤i ，至少需要添加几个数字。

思路：考虑某个位置添加数字时，会向后移动，因此模拟填入的数字直到刚好即可。

void Solve() {
	int n;
	cin >> n;
	vector<ll> a(n);
	for (int i = 0; i < n; i ++) cin >> a[i];
	ll ans = 0;
	for (int i = 0; i < n; i ++) {
		if (i + 1 + ans < a[i]) {
			ans += a[i] - (i + 1 + ans);
		}
	}
	cout << ans << '\n';
}

B

题意：给定一个数组，将数组分成多个区间数组。问是否有一种方案，使得分出来的所有区间数组的最长上升子序列长度异或和为 000 。

思路：对于偶数，每个元素直接划分为一个数组里即可，我们可以保证偶数个 111 的异或和为 000 。对于奇数长度数组，按照刚才方式划分会多出一个 111 ，这时候我们只需要找到一个长度为 222 的子数组，使得其最长上升子序列长度为 111 即可。

void Solve() {
	int n;
	cin >> n;
	vector<int> a(n);
	for (auto &i : a) cin >> i;
	if (n % 2 == 0) {
		cout << "YES\n";
		return;
	}
	for (int i = 1; i < n; i ++) {
		if (a[i] <= a[i - 1]) {
			cout << "YES\n"; return;
		}
	}
	cout << "NO\n";
}

C

题意：给定一个数组 aaa ，每次从数组中选择一个数字 aia_iai​ ，当满足 (i+1)∤ai(i + 1) \nmid a_i(i+1)∤ai​ 时，该数字可以去掉。问是否可以使得所有数字被去掉。

思路：对于第 iii 个位置上的元素 xxx ，如果它是可以被去除的，那么意味着在 [2,i+1][2, i+1][2,i+1] 的范围内一定存在一个值 jjj ，使得 j∤xj \nmid xj∤x 。若前面不存在 jjj ，使得 j∤xj \nmid xj∤x ，那么也就意味着 xxx 无法被去除掉。因此，我们可以对 aia_iai​ 检查其在 [2,i+1][2,i+1][2,i+1] 的范围内是否有数字不能够被整除。

此外，由于 lcm⁡(1,2,3,⋯ ,23)>109\operatorname{lcm} (1, 2, 3, \cdots , 23) > 10^9lcm(1,2,3,⋯,23)>109 ，已经大于 aia_iai​ 的范围了，因此也可以通过这个来加速判断。

void Solve() {
	int n;
	cin >> n;
	vector<int> a(n + 1);
	for (int i = 1; i <= n; i ++) cin >> a[i];
	for (int i = 1; i <= n; i ++) {
		bool ok{};
		for (int j = 2; j <= i + 1; j ++) {
			if (a[i] % j) {
				ok = true;
				break;
			}
		}
		if (!ok) {
			cout << "NO\n";
			return;
		}
	}
	cout << "YES\n";
}

D

题意：给定两个偶数 xxx 和 yyy ，求出任意一个 nnn ，使得 n mod x=y mod nn \bmod x = y \bmod nnmodx=ymodn 。

思路：我们可以尝试对求题目给定的 xxx 和 yyy 的大小做一点分类讨论。

当 x=yx = yx=y 时，显然我们可以直接让 n=xn=xn=x ，这样等式两边结果都为 000。

当 x>yx > yx>y 时，我们可以贪心地让右式的结果为 yyy ，那么让左式结果成立，nnn 要取 dx+ydx + ydx+y ，ddd 为正整数。简单起见，我们可以直接让 n=x+yn = x + yn=x+y 即可。

当 x<yx < yx<y 时，我们这时候发现很难直接简单地找到 nnn 的一个合适取值，那么我们讨论一下 nnn 相对于 xxx 与 yyy 的大小关系。

  • 假设 n<xn<xn<x ，那么 n mod x=nn \bmod x = nnmodx=n ，但是 y mod n<ny \bmod n < nymodn<n ，因此 n≥xn \ge xn≥x ；
  • 假设 n>yn>yn>y ，那么 y mod n=yy \bmod n = yymodn=y ，但是 n mod x<xn \bmod x < xnmodx<x ，又因为我们已经有前提条件 x<yx < yx<y ，因此 n≤yn \le yn≤y 。

综上所述，我们可以发现 x≤n≤yx \le n \le yx≤n≤y 。现在我们已经取得了 nnn 的取值范围，但是我们要如何确定 nnn 呢？我们先考虑一个特殊的数字： z=⌊yx⌋xz = \lfloor \cfrac{y}{x} \rfloor xz=⌊xy​⌋x 。显然， z \bdmo x = 0 成立，而 y mod zy \bmod zymodz 也会剩下一部分。题目保证是偶数，那么我们直接点，让 y−zy - zy−z 对半分，也就是 $n = z + \cfrac{y-z}{2} $ ，那么就可以保证左式得到一半、右式得到一半。

void Solve() {
	ll x, y;
	cin >> x >> y;
	if (x > y) {
		cout << x + y << '\n';
	} else if (x == y) {
		cout << x << '\n';
	} else {
		cout << y - (y % x) / 2 << '\n';
	}
}
